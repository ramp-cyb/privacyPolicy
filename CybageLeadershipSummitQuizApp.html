Cybage Leadership Summit Quiz App

Phone Status and Identity

  • App detects the presence of SIM card in phone and gets the cellular network state (Aeroplane mode, Data connection state).


    Wi-Fi connection information

  • App detects the Wi-Fi connection state and identity to prompt the user to connect to known wireless network which has the game server hosted and accessible.


The information is collected for App’s internal consumption only; it is not shared or published outside App. The collected state data is used to determine the correct game playing conditions and notify the user if they are not correct.
